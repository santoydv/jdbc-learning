# JDBC Course (PostgreSQL)

A basic Java project created to learn and practice **JDBC (Java Database Connectivity)** with **PostgreSQL**.  
This project demonstrates the complete JDBC workflow: driver loading, DB connection, executing queries, and closing resources.

---

## Features
- Connects Java application to PostgreSQL using JDBC
- Executes SQL queries using `Statement` / `PreparedStatement`
- Supports `SELECT` using `ResultSet` and `INSERT` operations
- Covers JDBC flow: Driver → Connection → Statement → Execute → Close

---

## Tech Stack
- Java
- JDBC
- PostgreSQL
